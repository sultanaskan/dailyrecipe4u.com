# 🍳 DailyRecipe4U Documentation
> A premium homemade recipe blogging platform.

Welcome to **DailyRecipe4U**, a comprehensive system designed for food lovers and culinary creators. This guide outlines user privileges and administrative operations.

---

## 👥 User Roles & Privileges

The system supports three distinct user roles, each with specific access levels.

### 1. Public User
* **Resource Access:** Can access all public resources.
* **Navigation:** Browse the entire website and search for recipes.
* **Read-Only:** View ingredients and cooking instructions.

### 2. Registered User
* **Includes all Public User privileges.**
* **Engagement:** Post star ratings and detailed reviews on recipes.

### 3. Administrator
* **Full System Control:** Access to all public and registered user features.
* **Content Management:** Full CRUD (Create, Read, Update, Delete) capabilities for recipes.

---

## 🛠 Administrator Guide

### 🔐 Accessing the Dashboard
1. Log in to your Admin account at [dailyrecipe4u.com](https://dailyrecipe4u.com).
2. Tap the **Profile** button in the top-right corner.
3. Scroll to the bottom of the Profile page and click **Go to Dashboard**.

### 📝 Content Management

#### **Adding a New Recipe**
1. Click **Add New Recipe** at the top of the Dashboard.
2. Fill out the recipe details (Title, Ingredients, Category, etc.).
3. **Featured Section:** To showcase this recipe on the homepage, check the **Featured** box at the bottom.
4. Click **Submit** to publish.

#### **Updating a Recipe**
1. Locate the recipe in the Admin Dashboard.
2. Click the **Edit** button.
3. Update the desired fields and click **Submit**.

#### **Deleting a Recipe**
1. Identify the recipe in the Dashboard.
2. Click **Delete**.
3. Confirm by clicking **Yes/OK** in the pop-up box.
   *⚠️ Note: This action cannot be undone.*
